\# SENG 384 Docker Assignment



\## Project Description



This project is a full-stack Person Management System built using React, Express.js, and PostgreSQL. The entire system is containerized using Docker and Docker Compose.



\## Technologies Used



\* Frontend: React (Vite)

\* Backend: Node.js (Express)

\* Database: PostgreSQL

\* Containerization: Docker \& Docker Compose



\## Features



\* Add a new person

\* View all people

\* Update person information

\* Delete a person



\## How to Run



docker compose up --build



Then open:

http://localhost:3000
